# Mathe-Quest Klasse 5 – GitHub-Bilder-Version

Diese Version ist für GitHub Pages gedacht. Die Missionsbilder werden extern aus demselben Ordner geladen wie die HTML-Datei.

## Dateien im gleichen Ordner ablegen

- `klasse5-lernspiele.html`
- `zahlen-groessen.webp`
- `rechnen.webp`
- `brueche.webp`
- `symmetrie.webp`
- `flaechen.webp`
- `volumen.webp`

## Wichtig

Die Dateinamen müssen exakt stimmen. GitHub unterscheidet Groß- und Kleinschreibung.

## Beispielstruktur

```text
klasse-5/
  lernspiele/
    klasse5-lernspiele.html
    zahlen-groessen.webp
    rechnen.webp
    brueche.webp
    symmetrie.webp
    flaechen.webp
    volumen.webp
```

Dann kann die HTML-Datei die Bilder mit relativen Pfaden laden, z. B. `zahlen-groessen.webp`.
