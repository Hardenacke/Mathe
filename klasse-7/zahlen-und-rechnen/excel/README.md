# Excel-Tabellen – Klasse 7

Dieser Ordner enthält den interaktiven Lernpfad zum Arbeiten mit Excel-Tabellen im Kontext eines Schulfest-Kiosks.

## Dateien

- `index.html`: finaler interaktiver Lernpfad mit 10 einfachen Excel-Aufgaben, Kontrollfragen, Tipps und Lösungen
- `Excel_Tabelle_Klasse7_Schulfest_Kiosk_mehrstufig.xlsx`: Arbeitsdatei für die Lernenden
- `assets/bilder/`: Visualisierungen für den Lernpfad
- `material/`: Arbeitsblatt, Kontrolllösung und Einsteigerübersicht als PDF und LaTeX

## Einbindung

Der Lernpfad ist für GitHub Pages vorbereitet und kann über folgenden relativen Pfad eingebunden werden:

`klasse-7/daten-und-zufall/excel/index.html`
